# Jump Banking System (Console Application)

A Java-based banking application built using object-oriented programming principles. This project simulates a banking system where users can create accounts, deposit and withdraw funds, transfer money, and administrators can manage customer accounts through a command-line interface.

## Features

### Customer Features
- Create checking and savings accounts
- Deposit funds
- Withdraw funds
- Transfer money between checking and savings
- View account balances
- View transaction history

### Administrator Features
- Secure administrator login
- View all customer accounts
- Create customer accounts
- Delete customer accounts
- Transfer money between customer accounts
- Search customers by username
